\# Tic-Tac-Toe (Django Project)



This is a simple \*\*Tic-Tac-Toe game\*\* built using the \*\*Python Django\*\* framework.  

The player uses \*\*X\*\*, and the computer uses \*\*O\*\*.  

The project was created as part of learning exercises, so some parts of the code were written independently and some during coursework.



\## 🔧 Features



\- Player vs Computer gameplay

\- Computer makes random valid moves

\- Win detection:

&nbsp; - horizontal

&nbsp; - vertical

&nbsp; - diagonal

\- Draw detection

\- Ability to restart the game

\- Simple and clean UI using HTML + CSS
